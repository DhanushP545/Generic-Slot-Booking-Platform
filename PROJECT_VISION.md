#Project Vision
##What this platform does?
- It enables users to book slots for generic resources, which can be specified to a limited scope and a single use case in the future.

##Roles in the project
- Provider : Can create resources, and make the slots available.
- Consumer : Can browse and book the slots for a particular resource.
- Administrator : Has a greater control over the resources, slots, and slots booked.

##Example Use Cases
- Turf owners can make slots available for different sport facilities that they can provide.
- Clinic staff can make the doctor appointments hassle free, by letting the patients book the available slots, instead of waiting at the clinic for hours.
- Tutors with multiple students, can give the flexibility to students to book the slot, instead of a rigid time block decided by the tutor.
